# ふるさと納税 検索システム 🌾

**使用技術：PHP / JavaScript / HTML / CSS**

## 概要

全国自治体の返礼品をキーワード・カテゴリー・寄附金額で検索できるWebアプリです。  
株式会社エッグが手掛ける「ふるさと納税システム」をイメージして作成しました。

## 機能一覧

- 🔍 キーワード検索（返礼品名・自治体名・説明文）
- 📂 カテゴリーフィルター（肉類・魚介類・果物・乳製品・体験など）
- 💴 寄附金額の上限フィルター
- 🔄 金額の並び替え（安い順・高い順）
- 📋 返礼品の詳細モーダル表示
- 📱 スマホ対応（レスポンシブデザイン）

## 画面イメージ

> ※ ブラウザで index.php を開くと以下の画面が表示されます

| 検索画面 | 詳細モーダル |
|---------|------------|
| キーワード・カテゴリー・金額で絞り込み可能 | カードクリックで詳細ポップアップ表示 |

## 使用技術

| 技術 | 用途 |
|------|------|
| PHP | サーバーサイド処理・検索フィルター・データ管理 |
| JavaScript | ソート機能・モーダル表示・動的UI |
| HTML / CSS | 画面レイアウト・レスポンシブデザイン |

## 動作環境

- PHP 7.4以上
- Webサーバー（Apache / Nginx）または `php -S localhost:8000` で起動可

## ローカルでの起動方法

```bash
git clone https://github.com/[あなたのID]/furusato-search.git
cd furusato-search
php -S localhost:8000
```

ブラウザで `http://localhost:8000` を開く。
